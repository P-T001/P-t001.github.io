- 数据恢复

```
R-studio
WinHex
Windows File Recovery ：（下载地址：https://www.microsoft.com/zh-cn/p/windows-file-recovery/9n26s50ln705?activetab=pivot:overviewtab）
	恢复Users下的jpg和png图片C: ->E:/outfile：  winfr C: E:/outfile /n \Users\*.JPEG /n \Users\*.PNG
	恢复pdf和docx文件C->E盘： winfr C: E: /r /n *.pdf /n *.docx
	恢复带invoice的文件：winfr C: E: /r /n *invoice*
```
